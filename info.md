# Anleitung: Testen des Frontends

Diese Anleitung erklärt Schritt für Schritt, wie man das Frontend des Projekts lokal aufsetzt und testet.

## Voraussetzungen

Bevor Sie beginnen, stellen Sie sicher, dass Folgendes installiert ist:
- Node.js (>=14.x)
- Ein Paketmanager (npm oder yarn)
- Ein Browser Ihrer Wahl

## Schritt 1: Projekt klonen
Klonen Sie das Repository auf Ihren lokalen Computer:
```bash
git clone https://github.com/ChristineJanischek/python-algorithmen-datenstrukturen.git
```

Wechseln Sie danach in das Verzeichnis des Frontend-Projekts:
```bash
cd python-algorithmen-datenstrukturen/apps/web
```

## Schritt 2: Abhängigkeiten installieren
Installieren Sie die benötigten Abhängigkeiten:
```bash
npm install
# oder wenn Sie yarn verwenden
yarn install
```

## Schritt 3: Entwicklungs-Server starten
Starten Sie den Entwicklungs-Server:
```bash
npm run dev
# oder wenn Sie yarn verwenden
yarn dev
```

## Schritt 4: Frontend im Browser öffnen
Öffnen Sie Ihren Browser und navigieren Sie zu:
```
http://localhost:3000
```
Dies öffnet das React/Vite-Frontend des Projekts.

## Schritt 5: Fehlerbehebung
Falls Probleme auftreten:
- Prüfen Sie die Konsolenausgabe auf Fehler.
- Stellen Sie sicher, dass alle Abhängigkeiten korrekt installiert sind.

# Nächste Schritte
