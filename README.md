## Über das Projekt
Das Ziel dieses Projekts ist es, Lehrern und Schülern die Möglichkeit zu geben, interaktiv Konzepte der Informatik zu verstehen und direkt anzuwenden. Es dient als Vorlage für erweiterbare E-Learnings.

### Highlights
- **Backend**: FastAPI
- **Frontend**: React/Vite
- **JSON-Kataloge**: Themen, Meilensteine und Aufgaben
